Description

This project implements a knowledge base for a large extended family.  The Royal British Family was used in this project from Queen Elizabeth II and King William. Down to the Fourth Generation.

It includes:
- 24 individuals from the Royal British family
- 4 generations
- Properties (gender, age, height)
- Basic relationships (mother, father, married)
- Derived relationships (brother, sister, sibling, husband, wife)
- Rules for comparing ages, heights, and identifying qualified individuals

The system supports:

1. Adding new members
2. Adding new properties
3. Querying individual properties
4. Listing all properties of an individual
5. Finding individuals with given property constraints
6. Querying family relationships
7. Listing all relationships a person has
